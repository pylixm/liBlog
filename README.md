# liBlog

一个基于Django1.11TLS版本的博客系统。自己的练手周边项目，会随时尝试新技术。

线上地址: [http://pylixm.top](http://pylixm.top)

开发环境：
- python3.6
- django1.11TLS
- mysql5.7

## 规划主要功能

- [x]博文章的草稿、发布和删除
- [x]分类查看文章
- [x]Tag 标签管理
- [x]文章评论
- [x]社会化登录，目前支持：
    - [x]微博
    - [x]github
- [x]支持将MarkDown文件导入
- []MarkDown文件的导出
- []专题系列分类
- []资讯头条
